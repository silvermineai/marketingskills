# Ahrefs Filter Cheat Sheet

Quick reference for filtering keyword results in Ahrefs to eliminate junk and surface high-value targets.

## Keyword Explorer Filters

### Exclude Words (Junk Removal)

Always exclude these unless the blog specifically targets them:

| Category | Words to Exclude |
|----------|-----------------|
| Low-value intent | free, cheap, craigslist, ebay, amazon, walmart |
| Outdated | 2020, 2021, 2022, 2023, 2024 (keep current and next year) |
| File types | pdf, download, torrent, template download |
| Platform-specific | reddit, quora, youtube, tiktok (unless targeting these) |
| Job-related | salary, jobs, hiring, careers |
| Academic | thesis, dissertation, assignment, homework |

### Include Words (Intent Forcing)

Use these to surface keywords with clear blog intent:

| Intent Type | Include Words |
|-------------|--------------|
| Informational | how, what, why, when, guide, tutorial, explained, examples |
| Commercial investigation | best, top, vs, comparison, review, alternative, recommended |
| Problem-solving | fix, solve, troubleshoot, prevent, avoid, mistake |
| Learning | learn, understand, beginner, advanced, step-by-step |

### KD Thresholds by Blog Stage

| Stage | Max KD | Min Volume | Word Count Min |
|-------|--------|-----------|---------------|
| New (< 6 months) | 15 | 50 | 3 |
| Growing (6-18 months) | 30 | 100 | 3 |
| Established (18+ months) | 50 | 100 | 2 |

## Content Gap Filters

When running content gap against competitors:

1. Set same KD and volume filters as above
2. Filter to keywords where **2+ competitors rank** (validates demand)
3. Exclude branded keywords (competitor brand names)
4. Sort by volume descending after filtering

## Batch Analysis

When vetting an imported keyword list:

1. Paste up to 10,000 keywords
2. Apply KD filter immediately
3. Sort by "Traffic Potential" (not just volume) — this accounts for SERP features
4. Flag keywords with very low CPC ($0.00) — may indicate no commercial value
